# Contexto Geral do Projeto - NextBot Chat

Este diretório serve como a base de conhecimento e referência de contexto para este projeto. Ele contém documentação estruturada sobre tarefas, decisões de implementação, resumos de sessões e a pilha tecnológica utilizada.

## Estrutura de Pastas

- **[tasks/](./tasks/)**: Acompanhamento de tarefas pendentes e concluídas.
- **[relatorios_implementacao/](./relatorios_implementacao/)**: Detalhes técnicos e decisões tomadas durante as fases de desenvolvimento.
- **[resumo_sessao/](./resumo_sessao/)**: Log de progresso de cada sessão de trabalho.
- **[stack/](./stack/)**: Documentação da pilha tecnológica, bibliotecas e dependências.
- **[onboarding.md](./onboarding.md)**: Guia rápido para novos desenvolvedores (ou para reinício de contexto).

## Objetivo
Centralizar a documentação viva do projeto para garantir que tanto desenvolvedores humanos quanto assistentes de IA tenham clareza sobre o estado atual, os objetivos e as restrições do sistema.
